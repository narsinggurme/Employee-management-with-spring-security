# Employee Management Backend

This is a **Spring Boot** backend application for managing employees. It provides RESTful API endpoints to **create, read, update, and delete** employee records, with **CORS support** for frontend integration.

---

## 🔹 Features

- RESTful API endpoints for managing employees
- CORS enabled for frontend communication (`http://localhost:4200`)
- CRUD operations: Create, Read, Update, Delete
- JSON-based request and response payloads
- Ready for integration with any frontend application

---

## 🔹 Technologies Used

- Java 17+
- Spring Boot
- Spring Data JPA
- Spring Web
- H2/MySQL (or any database of your choice)
- Maven

---

## 🔹 Getting Started

### Prerequisites

- Java 17 or higher
- Maven
